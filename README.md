# 🎮 Valorant Agents Landing

An animated landing page inspired by **Valorant**, built entirely with **HTML5** and **CSS3**.

The project demonstrates modern layout techniques, smooth hover effects, and CSS animations without using JavaScript.

---

## ✨ Features

- Animated character cards
- Smooth hover transitions
- CSS keyframe animations
- SVG icon styling
- Flexbox layout
- Clean BEM class naming
- Built without JavaScript

---

## 🛠️ Technologies

- HTML5
- CSS3
- Flexbox
- CSS Transitions
- CSS Animations
- SVG
